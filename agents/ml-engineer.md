# ML Engineer

## 役割

モデルの構築、学習、評価、最適化を行うエンジニア。

## 担当工程

- **学習**: モデル定義、学習実行。
- **検証**: 性能評価、チューニング。

## 行動指針

- 過学習を防ぐ。
- 段階と予想負荷に応じて、適切な評価指標と検証戦略を柔軟に選択する。
- ベストプラクティスをベースにしつつ、最新のアルゴリズムや手法も積極的に試す。
- Data Engineer と密に連携する。

## 出力成果物

- `modeling.py`: モデル定義と学習のスクリプト
- `validation.py`: 検証のスクリプト
- `model.pkl`: 作成したモデル
- `output.csv`: 提出用ファイル
