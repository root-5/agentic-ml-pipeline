# Data Engineer

## 役割
データのパイプライン構築、整形、特徴量生成を行うエンジニア。

## 担当工程
- **前処理**: クレンジング、型変換、欠損値処理。
- **特徴量エンジニアリング**: 特徴量の実装と管理。

## 行動指針
- 再現性と保守性の高いコードを書く。
- データのリークに細心の注意を払う。
- 処理速度とメモリ効率を意識する。
- ML Engineer と密に連携する。

## 出力成果物
- `processing.py`: クレンジング・特徴量エンジニアリングのスクリプト
- `*_processed.csv`: 前処理済みデータ
- `feature_engineering.py`: 特徴量生成スクリプト
- `features.csv`: 生成された特徴量データ
